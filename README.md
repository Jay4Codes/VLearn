# VLearn

SETUP:<br>
Clone the repo<br>
 `git clone https://github.com/Jay4Codes/VLearn.git`

Install all dependancies<br>
 `npm i`

Launch the react app on localhost<br>
 `npm start`

Build -> Visualize -> Explain Algorithms
